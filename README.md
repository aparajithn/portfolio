# Aparajith Nagarajan - Personal Portfolio

A modern, elegant personal portfolio website showcasing my journey as a founder, coffee enthusiast, and adventurer.

## About

This portfolio highlights my work, passions, and interests — from founding [Dabara Coffee](https://dabaracoffee.ca) to exploring the Rocky Mountains on two wheels.

## Features

- **Dark/Warm Aesthetic**: Inspired by coffee culture and Tamil heritage
- **Fully Responsive**: Beautiful on all devices
- **Fast & Modern**: Built with Next.js 14 and Tailwind CSS
- **Smooth Animations**: Engaging user experience with subtle transitions

## Sections

- **Hero**: Eye-catching introduction
- **About**: My story and background
- **Projects**: Featuring Dabara Coffee and future ventures
- **Interests**: Coffee, motorcycles, hiking, badminton, gaming, content creation
- **Contact**: Easy ways to connect

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (icons)
- Deployed on Vercel

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in your browser.

## Build

```bash
npm run build
npm start
```
